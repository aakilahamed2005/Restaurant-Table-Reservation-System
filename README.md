# 🍽️ Restaurant Table Reservation System

A complete Restaurant Table Reservation Management System developed using **Spring Boot** and **MySQL**.  
This system allows customers to reserve restaurant tables online while helping restaurant owners and administrators efficiently manage reservations, restaurants, tables, users, and customer feedback.

---

# 📌 Features

## 👤 User Management
- Customer Registration & Login
- Restaurant Owner Registration & Login
- Role-Based Authentication
- Profile Management
- Account Activation & Deactivation

## 🍴 Restaurant Management
- Add New Restaurants
- Update Restaurant Details
- Upload Restaurant Information
- Manage Restaurant Availability
- Restaurant Search & Viewing

## 🪑 Table Reservation
- Book Tables Online
- View Reservation History
- Manage Reservation Status
- Cancel Reservations
- Reservation Approval System

## ⭐ Customer Feedback System
- Add Reviews & Ratings
- Customer Comments
- Restaurant Feedback Management

## 🛠️ Admin Management
- Manage Users
- Manage Restaurants
- Monitor Reservations
- Handle Support Requests
- System Administration Features

## 🤖 Chatbot Support
- Integrated AI Chatbot
- User Assistance & Support
- Customer Interaction Handling

# 🏗️ System Architecture

The project follows a layered architecture:

- **Controller Layer** → Handles HTTP Requests
- **Service Layer** → Business Logic
- **Repository Layer** → Database Operations
- **Model Layer** → Entity Classes
- **DTO Layer** → Data Transfer Objects
- **Exception Handling Layer** → Custom Exceptions
